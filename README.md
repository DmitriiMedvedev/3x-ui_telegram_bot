# Dobrinya VPN Bot v17.1 (Stable Multi-Panel Proxy)

## Что нового (v17.1)

Бот полностью переработан для обеспечения максимальной стабильности в современных окружениях (Ubuntu 24.04/26.04, Python 3.14) и идеальной совместимости с VPN-клиентами.

### Ключевые изменения:
1. **Гибридный сервер подписок (Proxy Mode):** `sub_server.py` теперь работает как прокси-агрегатор. Он запрашивает нативные подписки напрямую у панелей 3X-UI, объединяет их и отдает клиенту. Это гарантирует 100% совместимость ссылок с любыми VPN-клиентами (v2rayNG, Streisand, Nekobox и др.).
2. **Синхронизация Inbounds в один клик:** В меню настроек сервера добавлена кнопка "Обновить из 3X-UI". Бот сам скачает все настройки (Reality, spiderX, flows, TLS) через API, избавляя от ручного копирования JSON.
3. **Резервный генератор (Fallback):** Если панели временно недоступны, сервер подписок автоматически переключается в режим генерации ссылок на основе данных из локальной БД. Поддерживаются современные параметры: `xtls-rprx-vision` (flow) и `spiderX` (spx).
4. **Авто-маппинг ID:** Исправлена проблема с импортом. Если в 3X-UI используется текстовый `tag` вместо числового `id`, бот автоматически найдет реальный ID через API.
5. **Совместимость с Python 3.14:** Манифест зависимостей актуализирован для работы с новым Stable C-API (aiogram 3.15+, pydantic 2.10+).

## Быстрый старт

### 1. Настройка окружения (`.env`)
Скопируйте `.env.example` в `.env` и заполните токены:
- `BOT_TOKEN`
- `SUB_BASE_URL` (например, `http://ваш_ip:8080/sub`)

### 2. Установка
```bash
cd /root/dobrinya_bot/3x-ui_telegram_bot
python3.14 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 3. Запуск
Бот настроен на рабочую директорию `/root/dobrinya_bot/3x-ui_telegram_bot/`.
```bash
cp dobrinya-bot.service /etc/systemd/system/
cp dobrinya-sub.service /etc/systemd/system/
systemctl daemon-reload
systemctl enable --now dobrinya-bot dobrinya-sub
```

### 4. Настройка в Telegram
1. `/addserver` — добавьте панель (используйте **API Token** для лучшей стабильности).
2. Зайдите в `/admin` -> **Серверы** -> Выберите ваш сервер -> **🌐 Обновить из 3X-UI**. Бот сам подтянет все конфиги.
3. `/sync` — принудительно добавьте всех существующих пользователей на новые инбаунды.

## Команды администратора
- `/admin` — Главная панель управления.
- `/status` — Проверка связи со всеми панелями.
- `/sync` — Синхронизация всех пользователей.
- `/debugtraffic` — Сверка трафика (БД vs Панели).
- `/userinfo ID` — Профиль пользователя.
- `/setbalance ID СУММА` — Управление балансом.
